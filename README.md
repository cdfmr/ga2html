# ga2html

Convert xml files produced by [GReader-Archive](https://github.com/Aulddays/GReader-Archive) to html pages.

### Platforms

OS X, Linux & Windows.

### Requirement

expat

### How to build

g++ -o ga2html -O2 ga2html.cc -lexpat

### How to use

Execute ga2html without any parameter to get help.
